# Blue Arc OS — Project Plan

## Overview
A web-based operating system interface built with Next.js, React, and Tailwind CSS.

## Milestones

### Milestone 1: App Shell (Current)
- Next.js project scaffold with Tailwind CSS
- Desktop-like shell layout (sidebar, header/taskbar, main content area)
- Window management system (open, close, focus windows)
- App launcher / start menu
- Mock data for desktop icons and apps
- Basic routing structure
- No database — mock data only
- Simple, clean UI

### Milestone 2: Core Apps (Future)
- File Explorer app with mock file system
- Terminal/Command Prompt app with mock commands
- Settings / Preferences app
- In-memory state management
- Keyboard shortcuts

### Milestone 3: Multi-User & Auth (Future)
- User login screen
- User profiles and sessions
- Per-user desktop state
- Auth middleware

### Milestone 4: Persistence & Backend (Future)
- Database integration (SQLite / IndexedDB)
- Persistent file system
- Persistent settings
- Data sync

### Milestone 5: Polish & Extras (Future)
- Drag-and-drop windows
- Window minimize / maximize / resize
- Wallpaper customization
- Dark mode
- Animations and transitions
